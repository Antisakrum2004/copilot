## Что по факту сейчас есть в коде

**Каркас Electron-приложения с написанными (но не компилировавшимися) сервисами.** Код выглядит как полноценная реализация пайплайна audio→STT→LLM, но содержит критические баги и ни разу не запускался. Это не MVP и не alpha — это **синтактически корректный прототип, ожидающий первой сборки и первого запуска**.

## История версий

**v0.0.1-alpha**: Написан код всех модулей. Проект **УСПЕШНО СОБРАЛСЯ** (electron-vite build): main (387 модулей, 367KB), preload (2 модуля, 5.3KB), renderer (34 модуля, 231KB). Electron dev mode не запускается на сервере (нет X11/GTK — ожидаемо). Первый запуск возможен только на Windows/macOS/desktop Linux с GUI.
